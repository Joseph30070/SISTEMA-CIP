CIP - Sistema Web de Gestión Académica
📌 Descripción

CIP es un sistema web desarrollado para la administración de una academia de profesores, permitiendo gestionar procesos de certificación, pagos y acceso a recursos educativos de manera centralizada.

La plataforma facilita la organización académica y mejora la comunicación entre administradores, docentes y estudiantes mediante enlaces directos a canales de clases y materiales.

FUNCIONALIDADES PRINCIPALES

    *Gestión de profesores y alumnos
    
    *Proceso de certificación de docentes
    
    *Registro y control de pagos
    
    *Administración de cursos y servicios
    
    *Acceso directo a canales de clases (ej. Telegram o plataformas externas)
    
    *Panel administrativo centralizado

  
TECNOLOGIAS USADAS

    *PHP (Backend)
    
    *MySQL (Base de datos)
    
    *HTML, CSS, JavaScript (Frontend)
    
    *Bootstrap (Diseño responsivo)
  

  
-Instalación

              Clonar el repositorio:
              git clone https://github.com/tuusuario/cip-sistema.git

Importar la base de datos en MySQL.
Configurar el archivo de conexión a la base de datos.
Ejecutar el proyecto en un servidor local (XAMPP o similar).
